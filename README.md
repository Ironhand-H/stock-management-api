# 📦 Stock Management API

Backend REST API for managing products and stock, developed with **Java & Spring Boot**.

---

## 🚀 Project Overview

This project is a backend application designed to manage products, quantities, and pricing through a RESTful API.

It is part of my transition toward **Java backend development**, focusing on building clean architecture and understanding core backend concepts.

---

## ⚙️ Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven

---

## 🧩 Features

- Create, read, update and delete products (CRUD)
- Layered architecture:
  - Controller
  - Service
  - Repository
- Data validation
- Exception handling (GlobalExceptionHandler)
- DTO pattern for data transfer

---

## 🔌 API Endpoints (examples)

| Method | Endpoint | Description |
|--------|----------|------------|
| GET    | /api/products | Get all products |
| POST   | /api/products | Create a product |
